# Advancy AI Assessments

Static GitHub Pages site for Advancy AI training questionnaires.

The AI Charter assessment is based on `Advancy_AI_Usage_Charter_final.docx` and presents one question at a time. After each submission, the correction explains why every answer option is correct or incorrect under the charter.

The AI Usage Training assessment is based on the June 2026 training presentations and excludes AI Usage Charter questions. It uses the same 25-question, A-E, single-answer format with option-by-option explanations.

Score collection is private by design. The public pages do not expose CSV export and do not store participant-readable result files. Scores are submitted only to the configured secure collector endpoint in `window.quizConfig.scoreEndpoint`; if no endpoint is configured, submission fails closed and the participant is told to contact the organizer.
